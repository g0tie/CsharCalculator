# CsharpCalculator

<h3>Calcul</h3>
Entrer un calcul puis appuyer sur la touche "=" pour effectuer le calcul

<h3>Inversion</h3>
Pour inverser le signe d'un nombre, entrer ce nombre puis appuyer sur la touche "+/-"
